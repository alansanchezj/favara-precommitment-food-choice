# When choice matters: the asymmetric effects of precommitment implementation on healthy food choice

**Favara, M., Mihaylova, J. and Sánchez, A.**

## Overview

This repository contains the dataset and code (Stata) to replicate the results in the paper.

## Folder structure

```
.
├── Data/        # Dataset used in the analysis
├── Dofiles/     # Stata do-file to reproduce all tables and figures
└── Output/
    ├── Graphs/  # Figures generated by the do-file
    └── Tables/  # Tables generated by the do-file
```

> **Note:** `Data/` and `Output/` are excluded from this repository. You must create them locally before running the code (see below).

## Requirements

**Software:** Stata

**Stata packages** (install with `ssc install <package>`):
- `fre`
- `ietoolkit`
- `boottest`
- `floatplot`

## Replication steps

1. Create the following folder structure locally:
   ```
   Replication files/
   ├── Data/
   ├── Dofiles/
   └── Output/
       ├── Graphs/
       └── Tables/
   ```

2. Place `analytical_dataset_SONGS.dta` in `Replication files/Data/`

3. Place `experiment_songs.do` in `Replication files/Dofiles/`

4. Open `experiment_songs.do` and update the folder path at the top of the file to match your local directory

5. Install required Stata packages (if not already installed):
   ```stata
   ssc install fre
   ssc install ietoolkit
   ssc install boottest
   ssc install floatplot
   ```

6. Run `experiment_songs.do` — all tables and figures will be saved to `Output/Tables/` and `Output/Graphs/`
